# RussianWordleDiscord
Wordle in Russian on Discord
Доброго времени суток, что бы ваш бот работал коректно, вам нужно закинуть все 99 картинок с изображением букв на пустой/новый сервер в виде емодзи.
Далее вам нужно заменить код каждого емодзи следущим образом: пишите \ и за ним вставляеете емодзи, Дискорд автоматически выдаст вам код этого емодзи, потом вам нужно зайти в utils.py и заменить его на нужный.
